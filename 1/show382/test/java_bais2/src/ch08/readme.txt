
Thread란?

process : 프로그램이 실행되면 OS로 부터 메모리를 할당 받아
프로세스 상태가 된다.
thread : 하나의 프로세스는 하나 이상에 thread를 가지게 되고
실제 작업을 수행하는 단위를 thread라고 한다. 



//싱글 쓰래드 코드를 작성했다.

//Multi Threading 프로그래밍
여러개의 thread가 동시에 수행되는 프로그래밍, 마치 여러작업이 
동시에 진행되는 효과를 얻을 수 있다.

각 각에 Thread 자신만에 자원을 가질 수 있다.
또는 메인 쓰래드가 가지고 있는 자원도 공유 받아서 사용할 수 있다.
하지만 자원을 공유 받아서 사용하게 되면 의도치 않은 잘못된
수행결과를 만날 수 있기 때문에 동기화 처리라는 개념을 알아야 한다.
