

자바 I / O 스트림

네트워크에서 자료에 흐름이 물의 흐름과 같다는 비유에서 유래 되었다.
입출력이 사용되는 곳 : 파일, 키보드, 네트웍, 메모리 등 모든 자료가 입력되고 출력되는 곳

대상 기준 / 입력 스트림, 출력 스트림
자료에 종류 / 바이트 스트림 / 문자 스트림
기능 / 기반스트림, 보조 스트림

바이트 단위 스트림과 문자 단위 스트림
바이트 단위 스트림 : 동영상, 음악 파일, 실행 파일 자료를 읽고 쓸 때 사용
문자 단위 스틀림 : 바이트 단위로 자료가 던져 지는데 매번 처리할려면 인코딩, 디코딩 해야하므로, 한번에
2바이트 이상 처리하도록 구현된 스트림
- Http 통신 - 문자 단위 스트림으로 데이터를 주고 받는다.
