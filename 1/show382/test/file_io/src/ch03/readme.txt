
바이트 단위 입력 스트림, 바이트 단위 출력 스트림


InputStream

바이트 단위로 입력 스트림 (최상위 추상 클래스)

주요 하위 클래스
FileInputSteam : 파일에서 바이트 단위로 자료를 읽어 들인다.
ByteArrayInputStream : byte 배열 메모리에서 바이트 단위로 자료를 읽는다.

-주요 메서드
int read() : 입력 스트림으로 부터 한 바이트의 자료를 읽는다. (데이터가 더이상 없으면 -1을 반환)
int read(byte b[]) 입력 스트림으로 부터 배열 크기의 자료를 읽는다.
void close() : 입력 스트림과 연결된 대상 리소드를 닫는다.

OutputStream

- 바이트 단위 출력에 최상위 추상 클래스
- 많은 추상 메서드가 선언 되어 있고 이를 하위 스트림이 상속 받아 구현 되어 있다.

- 주요 하위 클래스
FileOutputStream : 파일에서 바이트 단위로 자료를 씁니다.

- 주요 메서드
int write() : 한 바이트를 출력 합니다.
int write(byte b[]) : byte[] 배열 크기 만큼 한번에 자료를 출력 합니다.
