소켓통신
1.포트에 개념을 잡자
port - 항구(한대의 컴퓨터가 여러 프로그램을 동시에 사용하고 있다. 각각에 통신을 사용하는 프로그램에 교착지를 나타낸다.)


포트 번호는 - 0 ~ 65535번 까지 정해져 있다.
0 ~ 1023 포트까지는 가능한 선점해서 컴퓨터가 알아서 사용하는 포트라서 1024번 포트 부터 사용하기를 권장한다.

소켓통신이란?
양 끝단에 포트번호를 달아서 byte Stream 으로 데이터를 주고 받는 것이 소켓 통신이다.
(http 통신도 소켓을 활용해서 사용하는 통신이다)
